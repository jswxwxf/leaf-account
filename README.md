# 叶子记账 (leaf-account)

一个基于 `vue-mini` + `tailwindcss` 构建的微信小程序记账应用。

## 简介

本项目是一个简洁、易用的微信小程序记账应用，旨在帮助用户轻松记录日常开支。

技术栈:

- **核心框架**: [`@vue-mini/core`](https://vuemini.org/)
- **样式方案**: [`weapp-tailwindcss`](https://tw.icebreaker.top/)
